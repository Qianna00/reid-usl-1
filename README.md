# ReID-USL

## Introduction

A toolbox for unsupervised person re-ID tasks. It works with PyTorch == 1.6.0 for now.

## Model zoo

Supported self-supervised methods:

- [ ] BYOL
- [x] SimCLR
- [x] SimSiam

Supported methods:

- [x] [MMCL (CVPR'2020)](configs/mmcl/README.md)
- [x] [SpCL (NeurIPS'2020)](configs/spcl/README.md)
